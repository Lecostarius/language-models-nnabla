# Neural language models
implementation of neural language models with Sony's NNabla

## Requirements
- NNabla v0.9.7
- Keras v2.1.2 (for preprocessing)
- Numpy v1.13.3
- tqdm v4.19.5

## Models
- Recurrent neural network language model

- Character-level neural language model
